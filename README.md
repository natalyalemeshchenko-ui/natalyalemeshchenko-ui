## Кейсы

**Sun Camp, летние лагеря на базе детских садов Sun School, летняя смена 2026.** С нуля собрала лендинг под локацию, подключила AmoCRM, запустила РСЯ-кампанию через API Яндекс.Директа. В помощью вайб-кодинга настроила автоматическую генерацию лендингов и рекламных кампаний для летних лагерей в других локациях. Задача — при поступлении заявки от сада-партнера на рекламу летнего лагеря быстро развернуть посадочную страницу и настроить рекламу, обеспечить стабильный поток заявок по целевой CPA. Дальше: разработка общей страницы под лагеря, повышение конверсии лендингов, тестирование креативов.

**Сан Скул, реклама франшизы.** Работаю с воронкой для лидов по франшизе: отдельные лендинги, сегментация в РСЯ, отдельная аналитика по источникам. Главный фокус — качество лида, а не просто объём заявок.

**Бережная психология, SMM.** Выстроила контент-пайплайн от поста эксперта до публикации в Telegram, Instagram, VK, блоге. Сейчас система: готовый пост превращается в JSON-пакет контента + картинки-карусели + публикация через postiz. Сократило время на упаковку с нескольких часов до минут.

## Стек

**Реклама и аналитика:** Яндекс.Директ (включая API v5), Метрика, когортный анализ, CPA / ROI / CTR.

**Данные и автоматизация:** Excel, Google Sheets, Python (pandas, requests), Claude Code + AI-агенты.

**Интеграции:** AmoCRM, postiz (мульти-платформенный постинг), fal.ai (генерация креативов).

**Контент:** лендинги, презентации PowerPoint, Telegram / Instagram / VK.

## Чем могу быть полезна

- Воронка под образование: первый контакт → заявка → продажа абонемента или путёвки.
- Снижение CPA в РСЯ: гипотезы, креативы, нормальная аналитика.
- Автоматизация рутины: отчёты, парсинг отзывов, мониторинг.
- Упаковка эксперта в соцсетях: контент-стратегия, SMM-автоматизация.
